## 1. Google Reviews in Success Stories Section

**Commits:** `604e7be`

### What Changed
The "Amazing Success Stories" section was completely revamped to display real Google Reviews from the clinic's Google Business Profile instead of hardcoded testimonials.

### Files Modified
- **`src/data/testimonials.js`** — Complete rewrite

  - Each review now includes: `name`, `rating` (1–5), `relativeTime`, `text`, `profilePhoto`
  - Added `googleRatingSummary` export with:
    - `averageRating: 4.75`
    - `totalReviews: 61`
    - `googleMapsUrl`: [Google Maps Link](https://maps.app.goo.gl/SQzLPJ1HgCwTp36DA)

- **`src/components/TestimonialSlider.jsx`** — Enhanced UI
  - Google Rating Badge at top: overall rating, star display, review count
  - "Write a review" CTA button linking to Google Maps listing
  - Review cards with gold star ratings (using `react-icons`)
  - Google "G" icon on each card
  - Profile photo support (falls back to initials avatar)
  - "Powered by Google Reviews" attribution footer

- **`src/index.css`** — 170+ lines of new styles
  - Google rating badge (`.google-rating-badge`)
  - Star ratings (`.star-filled`, `.star-empty`)
  - Treatment tags, review time, Google icon on cards
  - Mobile responsive adjustments

### How to Update Reviews
1. Visit the [Google Maps listing](https://maps.app.goo.gl/SQzLPJ1HgCwTp36DA)
2. Copy reviewer name, rating, text, and time
3. Add/edit entries in `src/data/testimonials.js` → `googleReviews` array
4. Update `averageRating` and `totalReviews` in `googleRatingSummary`

## 2. Transparent Navbar with Scroll Effects

**Commits:** `e3a54bf`, `590276e`, `d35f747`, `9a04244`

### What Changed
The navbar now starts **fully transparent** at the top of the page and transitions to a **solid white glassmorphism** style when the user scrolls past 50px. The logo is hidden at the top and smoothly fades in when scrolled.

### Files Modified
- **`src/components/Navbar.jsx`**
  - Added `scrolled` state with `useEffect` scroll listener (threshold: 50px)
  - Navbar gets `.navbar-scrolled` class dynamically on scroll
  - Logo `<img>` gets `.navbar-logo.visible` class on scroll (hidden by default)

- **`src/index.css`**
  - `.navbar`: `position: absolute`, `background: transparent`, no shadow
  - `.navbar-scrolled`: `position: fixed`, white background with `backdrop-filter: blur(12px)`, shadow, slide-down animation
  - `.navbar-logo`: `opacity: 0`, `visibility: hidden`, `translateY(-10px)` by default
  - `.navbar-logo.visible`: `opacity: 1`, `visibility: visible`, `translateY(0)` with smooth transition

### Behavior
| State | Background | Logo | Position |
|-------|-----------|------|----------|
| At top (default) | Transparent | Hidden | Absolute (below TopBar) |
| Scrolled (>50px) | White/blur glass | Visible (fade in) | Fixed to top |

### Customization
- **Scroll threshold**: Change `50` in `Navbar.jsx` → `setScrolled(window.scrollY > 50)`
- **Transition speed**: Edit `transition` values in `.navbar` and `.navbar-logo` CSS rules
- **Scrolled background**: Change `rgba(255, 255, 255, 0.95)` in `.navbar-scrolled`
