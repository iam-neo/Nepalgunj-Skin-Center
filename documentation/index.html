Project Overview

The Nepalgunj Skin Center website serves as the digital front door for Western Nepal's premier aesthetic clinic. Patients can explore services, read the health blog, view testimonials, and contact the clinic.

Key Goals

  • Inform: Provide detailed info on Hair Transplants, Laser treatments, etc.
  • Trust: Showcase Dr. Abhishek Arjel's credentials and patient success stories.
  • Educate: Blog with skincare tips and treatment information.
  • Convert: Drive bookings through prominent "Book Now" calls to action.
  • Perform: Fast load times and smooth animations with React SPA.

Setup & Deployment

Prerequisites

You need Node.js 18+ installed.

Local Development

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

The development server runs at http://localhost:5173 (or next available port).

Deployment to Vercel

  1. Push the project to GitHub.
  2. Connect the repository to Vercel.
  3. Vercel auto-detects Vite and deploys.
  4. The vercel.json handles SPA routing automatically.
Note: The vercel.json file contains rewrites to ensure all routes redirect to index.html for proper SPA behavior.

Adding New Blog Posts

To add a new blog article:

  1. Open src/data/blogs.js
  2. Add a new object to the blogPosts array with all required fields:
{
  id: 'url-safe-slug',           // Used in URL: /blog/url-safe-slug
  title: 'Article Title',
  excerpt: 'Short description',
  content: `<p>HTML content</p>`, // Use h3 for headings (auto-generates TOC)
  image: '/images/blog/photo.jpg',
  category: 'Skin Care',         // Must match existing categories
  tags: ['Tag1', 'Tag2'],
  author: 'Dr. Abhishek Arjel',
  authorAvatar: '/images/hosp/doctor.jpg',
  authorBio: 'Short author bio...',
  date: '2026-02-19',
  readTime: '5 min read',
  views: 0,
  commentsCount: 0,
  featured: false,                // Set true to show in hero section
  faq: [
    { question: '...', answer: '...' }
  ],
  comments: [
    { name: '...', date: '2026-02-19', text: '...' }
  ]
}

The blog listing and detail pages will automatically include it with all features (TOC, FAQ schema, comments, etc.).
